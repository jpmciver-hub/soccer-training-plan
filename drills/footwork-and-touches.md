# Footwork & Touches (Monday)

**Duration:** 60 minutes  
**Goal:** Build ball mastery and comfortable close-control touch  
**Equipment:** Ball, cones (10+), flat surface

---

## Warm-Up (10 min)

| Drill | Duration | Description |
|-------|----------|-------------|
| Juggling | 5 min | Alternating feet. Count and try to beat personal best. Record touches. |
| Toe Taps | 3 min | Ball stationary, alternating feet on top. Start slow, build to game speed. |
| Ball Rolls | 2 min | Sole of foot, roll ball side to side while stationary. Both feet. |

---

## Core Session (40 min)

### Block 1: Foundation Touches (15 min)

| Drill | Reps/Duration | Coaching Points |
|-------|---------------|-----------------|
| **Inside-Inside** | 3 x 1 min | Tap ball between feet using inside of both feet. Stay on toes, knees bent. |
| **Inside-Outside (same foot)** | 3 x 1 min each foot | Touch inside, then outside, same foot. Keep ball close. |
| **Pull-Back Turn** | 10 each foot | Stop ball with sole, pull back, turn 180. Accelerate out. |
| **Cruyff Turn** | 10 each foot | Plant foot beside ball, chop behind standing leg with inside of foot. |
| **Sole Roll Across Body** | 3 x 1 min | Roll ball across body with sole, receive with opposite foot inside. |

### Block 2: Moving Touches (15 min)

Set up a 20-yard channel with cones.

| Drill | Reps | Coaching Points |
|-------|------|-----------------|
| **Inside-only dribble** | 4 lengths | Both feet, alternating touches. Head up between touches. |
| **Outside-only dribble** | 4 lengths | Push with outside of foot. Small touches, close control. |
| **Sole roll forward** | 4 lengths | Roll ball forward with sole, both feet alternating. |
| **Combo: inside-outside-sole** | 4 lengths | Cycle through all three surfaces on each foot per length. |
| **Speed dribble** | 4 lengths | As fast as possible with ball under control. Push and sprint. |

### Block 3: Patterns & Combos (10 min)

Set up a cone square (5x5 yards).

| Drill | Reps | Description |
|-------|------|-------------|
| **Box touches** | 5 min | Dribble around the square using a different surface on each side (inside, outside, sole, laces). |
| **Figure 8** | 5 min | Weave figure-8 pattern around two cones, 3 yards apart. Alternate which foot leads each rep. |

---

## Cool-Down (10 min)

| Activity | Duration |
|----------|----------|
| Juggling challenge | 5 min - try to beat morning count |
| Static stretching | 5 min - quads, hamstrings, calves, hip flexors, groin |

---

## Progression by Phase

| Phase | Adjustment |
|-------|-----------|
| **Weeks 1-3** | Focus on form. Slow pace. Master each surface. |
| **Weeks 4-6** | Increase speed. Add head-up requirement (call out a number of fingers parent holds up). |
| **Weeks 7-9** | All drills at game speed. Add passive defender pressure during moving touches. |
| **Weeks 10-12** | Combine into free-form sequences. Create own combos. Timed challenges. |

---

## Touch Count

A full session at recommended reps produces approximately **600-800 touches**. Log in [Touch Tracker](https://jpmciver-hub.github.io/touch-tracker/).
